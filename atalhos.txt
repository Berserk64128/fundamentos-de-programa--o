📄 CTRL + C => Copia (a seleção ou a linha) 
📄 CTRL + X => Recorta 
📄 CTRL + V - Cola 
📄 CTRL + Z - Desfaz

📄 CTRL + F = Pesquisa o caracter/palavra informado
